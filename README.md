
# FE_Test Azhari Aziz

## 🌐 Deployment

## 🔮 Features
- Login
- Halaman Pemindahan Barang
- Popup list barang
- Tambah barang yang akan dipindahkan
- Popup list lokasi
- Tambah lokasi ke barang
 
## 🧰 Installation

Clone this project

```bash
git clone https://github.com/Azhari12/FE-Test.git
cd FE-Test
```
⌛ Install
```bash
npm install
```
🚀 Usage
```bash
npm run dev
```
